# TruthLens AI

AI-powered Deepfake Detection Platform built using Next.js, FastAPI, OpenCV, and AI-based image analysis.

## Live Demo

https://truthlens-ai-kappa.vercel.app

## Features

- AI-powered image authenticity analysis
- Deepfake prediction system
- OpenCV face detection
- Blur analysis
- AI confidence scoring
- JWT Authentication
- Responsive premium UI
- Full-stack cloud deployment

## Tech Stack

### Frontend
- Next.js
- React
- Tailwind CSS
- Axios

### Backend
- FastAPI
- Python
- OpenCV
- TensorFlow
- JWT Authentication

### Deployment
- Vercel
- Render
